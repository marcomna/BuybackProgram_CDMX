A continuación, se presenta la descripción de caracteres (siglas) para el manejo de valores que indican o representan algún estatus dentro de la información de los conjuntos de datos:

-	NSS: No cuenta con elementos para responder

	Se utiliza para identificar las variables en las que el informante, por falta de información, respondió no saber la respuesta; así como en aquellas variables que, por su carácter condicionado a otras respuestas, mantienen el mismo estatus.

-	NA: No aplica

	Se utiliza para identificar las variables que, por la naturaleza de la pregunta correspondiente, no aplica proporcionar información; así como en aquellas variables que, por su carácter condicionado a otras respuestas, no deben ser respondidas.

-	ND: Información no disponible

	Se utiliza para identificar las variables en las que, por diversas razones, el informante no respondió el cuestionario.

-	NP: No publicable por el principio de confidencialidad o por reserva de información

	Se utiliza para identificar las variables en las que el informante proporcionó información, pero que, por confidencialidad o reserva, no es susceptible de ser publicada.
